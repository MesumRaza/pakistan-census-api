# Pakistan Census API

## Running
1. ```virtualenv venv```
2. ```pip install -r requirements.txt```
3. ```python app.py```

## Methods

### Provinces

#### Get all provinces
GET ```/provinces```

### Districts

#### Get all districts
GET ```/districts```


### Top 10 Cities

#### Get top 10 cities in terms of population
GET ```/provinces```
